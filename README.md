# ميثاق — Mithaq Auth

نظام مصادقة مبسّط (تسجيل دخول / إنشاء حساب) يعمل بالكامل في المتصفح باستخدام HTML و CSS و JavaScript خالص — بدون أي مكتبات خارجية.

## ✨ المميزات
- إنشاء حساب، تسجيل الدخول، وتسجيل الخروج
- تشفير كلمة المرور بـ SHA-256 + Salt عبر Web Crypto API
- جلسة (Session) محفوظة محليًا تبقى بعد إعادة تحميل الصفحة
- لوحة تحكم تعرض بيانات المستخدم (الاسم، البريد، تاريخ الإنشاء، آخر دخول)
- واجهة عربية (RTL) نظيفة ومستوحاة من تصميم Clerk

## 🚀 التشغيل
افتح ملف `index.html` مباشرة في المتصفح، أو فعّل **GitHub Pages** على المستودع لرؤيته أونلاين.

## ⚠️ ملاحظة مهمة
هذا مشروع **تعليمي** فقط. تُخزَّن البيانات في `localStorage` داخل متصفحك ولا تُرسَل إلى أي خادم. لا تستخدمه لحسابات حقيقية أو بيانات حسّاسة.

للأنظمة الحقيقية على مستوى الإنتاج، تحتاج إلى:
- خادم (Backend) آمن مثل Node/Express
- قاعدة بيانات (MongoDB / PostgreSQL)
- إدارة جلسات ورموز آمنة (JWT)، تحقق بالبريد، حماية ضد الهجمات
- أو استخدام خدمة جاهزة مثل **Clerk** أو **Auth0**

## 📁 الملفات
- `index.html` — الهيكل والواجهة
- `style.css` — التنسيق والتصميم
- `script.js` — منطق المصادقة
